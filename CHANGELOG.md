## [2026-06-16]

### Added

It contains snapshot of "stable enough" but strictly reproducible Ubuntu 24.04 LTS image. It usually will be based on the latest Ubuntu 24.04 update (but not always) and it has some commonly used OS dependencies such as unzip, curl, etc that are frozen in time. They have exact version like in the day of the snapshot release.

Sometimes, only version of these OS dependencies will be updated, or new OS dependencies added.

Sometimes, also base ubuntu 24.04 image will be updated to latest.

See https://github.com/alex-ber/ubuntu2404-snapshots for source code.
